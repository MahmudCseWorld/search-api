# installation 
 
  1. clone the repo
  2. yarn install
  3. yarn start (it will run the server)

## endpoint 
    
    1. open your borwser 
    2. type in tab: http://localhost:5001/api/searchterm(search for any thing)

## example 
    you want to search for hello
    then put it after api/

     http://localhost:5001/api/hello
